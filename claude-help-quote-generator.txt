  What was built

  - Category filter buttons (All / Saints / Scripture / Inspiration)

  - Fade in/out animation on every quote change (0.4s CSS transition)
  - Gold cross symbol ✝ as the page icon
  - Decorative large " quotation mark on the card
  - Category badge pill on each quote
  - Button with lift + glow hover effect

  Design choices:
  - Warm cream background (#F6F1E9) with soft gold radial glows
  - White card with a 4px gold top border and deep shadow
  - Playfair Display (serif, italic) for quotes — elegant and readable
  - Lato (sans-serif) for UI elements
  - Gold accent color (#B8890A) throughout

  ---
  How to run it

  No build tools or server needed — it's plain HTML/CSS/JS.

  Option A — Double-click:
  Open index.html directly in any browser (Chrome, Edge, Firefox, Safari).

  Option B — VS Code Live Server:
  Right-click index.html in VS Code → Open with Live Server (auto-reloads on
  save).

  Option C — Terminal:
  # Python (most systems have it)
  python -m http.server 5500
  # Then open http://localhost:5500

  The Google Fonts (Playfair Display, Lato) load from the CDN, so an internet
  connection is needed for the full typography. The app works offline but will
  fall back to generic serif/sans-serif fonts.

  localStorage is a browser built-in that stores data as strings, permanently,
  even after closing the tab:

  // Save
  localStorage.setItem("catholicFavorites", JSON.stringify(favorites));

  // Load (on page open)
  let favorites = JSON.parse(localStorage.getItem("catholicFavorites") || "[]");

  The flow:
  1. User clicks ♡ Save → saveFavorite() pushes the quote into the favorites
  array and calls persistFavorites() to write it to localStorage
  2. renderFavorites() rebuilds the list using createElement() (safe, no HTML
  injection)
  3. Each item gets a × button that calls removeFavorite(quote.text) — it
  filters the array and re-renders
  4. The Save button auto-updates to ♥ Saved / ♡ Save whenever a new quote
  appears or a favorite is removed
